Explicar o problema:
    Cláudio gerencia seu mini mercado de forma manual, utilizando planilhas e um caderno para registrar pagamentos aos fornecedores e débitos dos clientes. Esse método tem causado falhas no controle financeiro, como atrasos no pagamento de boletos e esquecimento de cobrar clientes, o que resulta em juros pagos por atraso e perda de dinheiro.

Explicar a solução:
    Foi desenvolvido um sistema para automatizar o controle financeiro do Cláudio, permitindo o registro eficiente das contas a pagar e a receber. O sistema calcula automaticamente os juros das contas. Além disso, ele oferece a funcionalidade de registrar pagamentos parciais, permitindo que Cláudio controle de forma mais precisa as pendências e evite perdas financeiras. Com essa solução, Cláudio terá uma gestão mais organizada, reduzindo erros e atrasos nos pagamentos e cobranças.

Ferramentas usadas:
    HTML, CSS, JAVASCRIPT E FIREBASE. 

Como executar a solução criada:
   1. Abra a pasta do projeto no vscode.
   2. Na pasta "html" abra o arquivo que deseja executar (com o botão direito selecione: Open with Live server).
   3. Utilize o navegador para executar o programa.
   Obs: Para ter acesso aos dados do banco de dados, será necessário utilizar o firebase.
   As credenciais se encontram na pasta "js" e no arquivo "firebaseConfig.js".
